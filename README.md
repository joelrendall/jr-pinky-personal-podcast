# Pinky's Personal Intelligence Briefs

Personal podcast feed for AI-generated intelligence briefings.

## Feed URL
https://joelrendall.github.io/jr-pinky-personal-podcast/feed.xml

## Brief Types
- **OpenClaw Intelligence**: Community updates from X & Reddit
- **PP Company Brief**: Team activity from Slack & GitHub  
- **Competitive Analysis**: Market research & user insights
- **Custom Briefs**: On-demand research topics

## Technical Details
- Multi-voice audio with dramatic handoffs
- Auto-rotation maintains 50 most recent episodes
- RSS 2.0 with iTunes podcast extensions
- 2K resolution cover art for each brief type
